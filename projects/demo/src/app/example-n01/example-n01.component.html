Getting Started

  1. To install RedZoom, run the command below: npm install ngx-red-zoom
  2. Import module: import {{ '{' }} RedZoomModule } from 'ngx-red-zoom'; @NgModule({{ '{' }} imports: [ RedZoomModule ] }) export class AppModule {{ '{' }} }
  3. Import styles: @import '~ngx-red-zoom/styles/base.scss'; @import '~ngx-red-zoom/styles/style-window.scss';
  4. Add directive to your image tag: <img src="example.jpg" redZoom="example.full.jpg" redZoomClass="red-zoom--style--window" >
